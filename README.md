# Formulario de Contacto

Proyecto web desarrollado con HTML5 y CSS3 como práctica de estructura semántica, estilos y organización de archivos.

## Descripción

Este proyecto consiste en un formulario de contacto responsive que permite ingresar datos personales y enviar una consulta.

El objetivo es practicar la creación de interfaces web utilizando buenas prácticas de HTML y CSS.

## Características

- Estructura HTML5 semántica.
- Formulario con campos de texto, email, teléfono, selección y mensaje.
- Validaciones básicas mediante atributos HTML5.
- Diseño adaptable para dispositivos móviles.
- Uso de clases organizadas con metodología BEM.
- Estados visuales con pseudoclases CSS.

## Tecnologías utilizadas

- HTML5
- CSS3
- Git
- GitHub
