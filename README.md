## Funcionalidades

- Formulario responsive.
- Validaciones HTML5 y JavaScript.
- Estilos desarrollados con CSS/SASS.
- Organización mediante metodología BEM.
- Diseño adaptable con Flexbox.

## Archivos principales

- index.html → estructura del sitio.
- css/styles.css → estilos compilados.
- scss → archivos fuente SASS.
- js/validation.js → validaciones del formulario.
